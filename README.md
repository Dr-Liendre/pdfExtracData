# pdfExtracData

Este proyecto consiste en la extraccion de informacion de una zona concrea del PDF, el pie de página. 
Es muy simple, lo que hacemos es leer todos los documentos pdfs que esten en una carpeta(no si hay subcarpetas=, y extraemos la informacion que nos interesa en un archivo excel. En este proyecto leemos el pie de pagina que contientiene la informacion de la revision en que se encuentra dicha pagina del documento y el capitulo(del indice) al que pertenece este documento.
